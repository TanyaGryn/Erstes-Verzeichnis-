Drei Möglichkeiten der Positionierung von Text- und Bildelmenten auf Websiten - Normaler Fluss: Position eines Elements wird durch die Reihenfolge im Dokument und die Art der Darstellung (z.b. block) bestimmt - Absolute Positionierung: Elemente vollständig aus dem normalen Fluss herausgenommen und unabhänig von anderen Elementen positioniert - Float Eigenschaft: ___2. Definition "Float Eigenschaft": - Ein Float nimmt ein Element aus dme normalen Fluss heraus - Nachfolgende Boxen verhalten sich, als wäre das Float-Element nicht vorhanden, lediglich die Inhalte weichen der Float - Box aus. ___3. Mögliche Werte: ___3.1 "left" Element wird möglichst weit links platziert und die Inhalte, die folgen, fließen rechts daran vorbei ___3.2 "Right" Element wird möglichst weit rechts platziert und die Inhalte, die folgen, fließen links daran vorbei ___3.3 "None" Das Element wird nicht umflossen und folgt dem noramlen Textfluss. ___4. Darstellung: Elemente mit float -Eigenschaften werden als Blockelemente dargestellt und können Breite und Höhe zugewiesen bekommen ___5. Float Eigenschaften: ___5.1 Clear: Mit der clear-Eigenschaft wird das Umfließen vorheriger gefloateter Elemente beendet (clear left, clear right, clear both, clear none) ___5.2 Eingeschlossene Floats: Floatswerden nicht vom umgebenden Element z.B. rote Box umflossen, und ragen deshalb hinaus ___5.3 Floats umschließen mit clear ___5.4 Floats umschließen mit float ___6. Versteht man die Float Eigenschaft, kann man jede beliebige Website interessant gestalten. Der Komposition sind dabei keine Grenzen gesetzt. DANKE !
